# Portail vincent.bichat.fr

Portail personnel de Vincent Bichat vers ses sites, projets clients et side-projects.

## Développement

```bash
npm install
npm run dev
```

## Production

```bash
npm run build
npm run start
```

## Structure

- `src/app` : routes App Router, métadonnées et fichiers SEO
- `src/components` : composants UI réutilisables
- `src/data` : contenu centralisé pour les sites, profils et liens
- `public` : assets statiques, dont l'image de partage
